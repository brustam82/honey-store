# 🍯 Telegram-бот магазина мёда и пчелопродукции

## Структура файлов

```
honey_bot/
├── bot.py              — основной код бота
├── config.py           — настройки (токен, ID админа, таблица)
├── credentials.json    — ключ Google Service Account (см. ниже)
├── requirements.txt    — зависимости Python
└── README.md
```

---

## Шаг 1 — Создайте бота в Telegram

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте `/newbot`
3. Придумайте имя и username бота
4. Скопируйте **токен** — вставьте в `config.py` в строку `BOT_TOKEN`

---

## Шаг 2 — Узнайте свой Telegram ID

1. Напишите боту **@userinfobot** команду `/start`
2. Он пришлёт ваш `id` — вставьте в `config.py` в строку `ADMIN_ID`

---

## Шаг 3 — Настройте Google Таблицу

### 3.1 Создайте таблицу
Создайте новую Google Таблицу и добавьте **2 листа**:

**Лист "Товары"** — заголовки в строке 1:
| Название | Описание | Цена | Фото | Категория | Активен |
|----------|----------|------|------|-----------|---------|
| Мёд гречишный | Тёмный, насыщенный | 450 | https://... | Мёд | да |
| Прополис 10г | Натуральный | 200 | https://... | Прополис | да |

**Лист "Заказы"** — заголовки в строке 1:
| Дата | ID пользователя | Username | Имя | Телефон | Товар | Кол-во | Цена | Сумма |

### 3.2 Скопируйте ID таблицы
Из URL таблицы: `docs.google.com/spreadsheets/d/**ВОТ_ЭТО**/edit`
Вставьте в `config.py` в строку `SPREADSHEET_ID`

### 3.3 Создайте Service Account (ключ доступа)

1. Перейдите на [console.cloud.google.com](https://console.cloud.google.com)
2. Создайте новый проект
3. Включите APIs: **Google Sheets API** и **Google Drive API**
4. Перейдите в **IAM и администрирование → Сервисные аккаунты**
5. Нажмите **Создать сервисный аккаунт**
6. Назовите его, нажмите **Готово**
7. Нажмите на созданный аккаунт → **Ключи → Добавить ключ → JSON**
8. Скачанный файл переименуйте в `credentials.json` и положите рядом с `bot.py`
9. Скопируйте email сервисного аккаунта (например: `bot@project.iam.gserviceaccount.com`)
10. В Google Таблице нажмите **Поделиться** и добавьте этот email с правами **Редактор**

## Возможности бота

## Поддержка

Если что-то не работает — проверьте:
1. Верный ли токен бота в `config.py`
2. Верный ли ваш ADMIN_ID
3. Добавлен ли email сервисного аккаунта в таблицу
4. Правильно ли названы листы ("Товары" и "Заказы")
